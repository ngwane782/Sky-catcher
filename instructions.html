Sky Catcher

Game Instructions

Game Goal

Objects fall from the sky and the player must move the bat left and right to catch them before they hit the ground.

Each caught object increases the score. Each missed object reduces the player's lives.

How to Play

  1. Enter your player name on the launcher page.
  2. Select a difficulty level.
  3. Select a bat colour.
  4. Choose optional features such as double score or random object sizes.
  5. Click Open Game Window.
  6. In the new window, click Start Game.
  7. Move the bat using any of the following controls:
    • Left Arrow / Right Arrow keys on the keyboard
    • Mouse – move the cursor over the game area
    • Touch – drag your finger across the game area (mobile)
  8. Catch as many falling objects as possible before they reach the ground.

Difficulty Levels

  • Easy: wider bat, slower falling speed, slower object spawning
  • Medium: medium bat width, medium speed
  • Hard: narrow bat, faster falling speed, faster spawning

Combo Streak

Catch objects consecutively without missing to build a streak. A higher streak multiplies the points you earn:

  • 1–2 in a row: normal points
  • 3–4 in a row 🔥: 2× point multiplier
  • 5+ in a row 🔥🔥: 3× point multiplier

Missing an object resets your streak to zero.

Buttons in the Game Window

  • Start Game starts a new game
  • Pause / Resume pauses or resumes the game
  • Save Session stores the current game state
  • Load Session restores a saved session
  • Reset Game clears the current game
  • Back to Settings closes the game window and returns to the launcher

Scoring

  • Each caught object adds points
  • If Double score mode is enabled, each successful catch awards double points
  • Build a Combo Streak for extra multipliers
  • Missing objects reduces lives
  • The game ends when no lives remain
